# ui/settings/

Ecran de configuration des parametres du modele.

- **SettingsScreen.kt** - Ecran principal avec sections Sampling, Generation et Performance
- **ParameterSlider.kt** - Slider reutilisable pour les parametres float/int avec description et erreur
- **ParameterDropdown.kt** - Dropdown reutilisable (ex: taille du contexte)
- **HardwareInfoCard.kt** - Carte affichant les informations materielles (CPU, RAM, GPU/Vulkan, NPU)
- **OptimizationSuggestions.kt** - Suggestions d'optimisation avec bouton "Appliquer"
- **ModelSelectionDialog.kt** - Dialogue de selection de modele
