# ui/common/

Composants UI partages entre les ecrans.

- **TopAppBar.kt** - Barre superieure avec menu et actions
- **MessageInput.kt** - Champ de saisie de message avec bouton d'envoi
- **ModelSelector.kt** - Selecteur de modele GGUF
- **HistoryMenu.kt** - Liste des conversations dans le drawer lateral
