# ui/

Couche interface utilisateur en Jetpack Compose.

- **NavGraph.kt** - Navigation de l'application (Home, Chat, Settings) avec drawer lateral pour l'historique
- **chat/** - Ecran de chat et composants de messages
- **common/** - Composants partages (barre d'app, selecteur de modele, input)
- **home/** - Ecran d'accueil avec selection du modele
- **settings/** - Ecran de configuration des parametres
- **syntax/** - Coloration syntaxique pour les blocs de code
- **theme/** - Theme Material 3 (couleurs, typographie)
